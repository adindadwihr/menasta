MENASTA

An innovative solution to facilitate city governments or related agencies in efficiently tracking, managing, and maintaining their asset inventories.

Stack:
1. React Native
2. GoogleSheets
3. Apps Script
4. FontAwesome5
5. HTML
6. GitHub
7. LeafletJS

Sumber data:
